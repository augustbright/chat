A chatting application with rooms. 
* **Node.js** and **Express** for backend REST serving, 
* **Passport.js**, **Mongo store** for user authorization and sessions, 
* **React.js**, **Bootstrap**, **Sass**, **Typescript** for frontend, 
* **Next.js** for server rendering, 
* **Redux**, **Redux-saga** for state management, 
* **WebSockets** for client notifications.
